# 🎮 Game Action System

Entity-component system for game development with modular effects.

## Features
- Entity-component architecture
- Keyboard movement controls
- Wind effect simulation
- Color change effects
- Modular effect system

## Quick Start
```bash
pip install -r requirements.txt
python main.py
```

## Architecture
- `GameEntity`: Base game object
- `Effect` classes: Modular behaviors
- `KeyboardMoveEffect`: Player controls
- `WindEffect`: Environmental effects

## Controls
- Arrow keys or WASD for movement
- Effects apply automatically
