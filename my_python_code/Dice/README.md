# 🎲 Dice Simulator

Random dice rolling simulation with various dice types.

## Features
- Multiple dice types (D4, D6, D8, D12, D20)
- Multiple dice rolling
- Statistical analysis
- Roll history
- Probability calculations

## Quick Start
```bash
python dice.py
```

## Usage
- Select dice type
- Choose number of dice
- Roll and view results
- Analyze roll statistics
