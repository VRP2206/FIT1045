# 💰 Change Calculator V3

Advanced change calculator with utility functions.

## Features
- Modular utility functions
- Enhanced calculation accuracy
- Improved code organization
- Better error handling

## Quick Start
```bash
g++ -o change_v3 change_v3.cpp utilities.cpp
./change_v3
```

## Files
- `change_v3.cpp`: Main application
- `utilities.cpp`: Helper functions
- `utilities.h`: Function declarations
