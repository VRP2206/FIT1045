# 📊 Stats Calculator

Statistical analysis tool for calculating various statistical measures.

## Features
- Mean, median, mode calculations
- Standard deviation
- Data range analysis
- Multiple data input methods
- Results summary

## Quick Start
```bash
g++ -o stats stats.cpp
./stats
```

## Statistics Calculated
- Arithmetic mean
- Median value
- Mode (most frequent)
- Standard deviation
- Min/max values
