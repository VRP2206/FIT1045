# 🪰 Fly Catch Game

Real-time action game where players catch flies with scoring system.

## Features
- Real-time fly movement
- Click-to-catch mechanics
- Score tracking
- Timer-based gameplay
- SplashKit graphics

## Quick Start
```bash
g++ -o fly_catch fly_catch.cpp -lSplashKit
./fly_catch
```

## Gameplay
- Click on flies to catch them
- Score increases with each catch
- Flies move randomly around screen
- Time limit adds challenge
