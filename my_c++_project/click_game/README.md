# 🖱️ Click Game

Reaction-based clicking game with scoring and timing mechanics.

## Features
- Fast-paced clicking gameplay
- Reaction time measurement
- Score tracking
- Visual targets
- Performance metrics

## Quick Start
```bash
g++ -o click_game click_game.cpp -lSplashKit
./click_game
```

## Gameplay
- Click on targets as they appear
- Score based on accuracy and speed
- Reaction time tracking
- Progressive difficulty
