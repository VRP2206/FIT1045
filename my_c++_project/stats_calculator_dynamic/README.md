# 📊 Dynamic Stats Calculator

Enhanced statistics calculator using dynamic arrays for flexible data handling.

## Features
- Dynamic array implementation
- Variable dataset sizes
- Memory-efficient processing
- Advanced statistical functions
- Custom data structures

## Quick Start
```bash
g++ -o stats_dynamic stats_dynamic.cpp
./stats_dynamic
```

## Files
- `stats_dynamic.cpp`: Main application
- `dynamic_array.h`: Array header
- `dynamic_array.hpp`: Array implementation
