# 🪨 Rock Dodge

Dynamic obstacle avoidance game with moving rocks.

## Features
- Player movement controls
- Falling rock obstacles
- Collision detection
- Score system
- Increasing difficulty

## Quick Start
```bash
g++ -o rock_dodge rock.cpp -lSplashKit
./rock_dodge
```

## Controls
- Arrow keys or WASD for movement
- Avoid falling rocks
- Survive as long as possible
- Score increases over time
