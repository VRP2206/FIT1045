# 🗼 Tower of Hanoi

Classic puzzle game with graphical interface using SplashKit.

## Features
- Interactive tower puzzle
- Drag-and-drop mechanics
- Move validation
- Visual feedback
- Win condition detection

## Quick Start
```bash
g++ -o hanoi tower_of_hanoi.cpp -lSplashKit
./hanoi
```

## Rules
- Move disks from left tower to right tower
- Only smaller disks can be placed on larger ones
- Use middle tower as temporary storage
