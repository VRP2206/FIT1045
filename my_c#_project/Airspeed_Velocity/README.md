# ✈️ Airspeed Velocity Calculator

Physics-based airspeed calculation application using SplashKit.

## Features
- Travel distance and time input
- Average speed calculation
- Remaining time estimation
- SplashKit graphics interface

## Quick Start
```bash
dotnet run
```

## Usage
1. Enter your name
2. Input distance travelled (km)
3. Input time taken (minutes)
4. Input remaining distance (km)
5. View calculated results
