# 🏠 Scalable House Drawing

Advanced house drawing application with scalable graphics using SplashKit.

## Features
- Scalable architectural graphics
- Proportional house elements
- Dynamic sizing capabilities
- Clean geometric rendering

## Quick Start
```bash
dotnet run
```

## Scaling Features
- Adjustable house dimensions
- Proportional window/door scaling
- Responsive design elements
- Maintain architectural proportions
