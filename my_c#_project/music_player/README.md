# 🎵 Music Player

Audio playback application using SplashKit multimedia capabilities.

## Features
- Audio file playback
- Music control interface
- SplashKit audio integration
- User-friendly controls

## Quick Start
```bash
dotnet run
```

## Supported Formats
- WAV, MP3, OGG audio files
- Located in `../music_list/` directory

## Controls
- Play/Pause functionality
- Volume control
- Track selection
