# 🎯 2D Pathfinding Trial 5

Enhanced 2D pathfinding with interactive grid examples.

## Features
- A*, Dijkstra, RRT algorithms
- Interactive grid manipulation
- Performance comparison
- Custom obstacle placement

## Quick Start
```bash
streamlit run app.py
```

## Files
- `pathfinding.py`: Algorithm implementations
- `grid2d.py`: Grid data structure
- `comparison.py`: Performance analysis
- `interactive_grid_example.py`: Grid interaction demo
