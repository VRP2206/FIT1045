## Getting Started

### Prerequisites
- Python 3.8+ with pip
- .NET 9.0 SDK
- C++ compiler (GCC/MSVC)
- SplashKit SDK

### Quick Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd mycode
   ```

2. **Python Projects**
   ```bash
   cd FIT1045_PROJECT/HD_Project/trial1
   pip install -r requirements.txt
   streamlit run app.py
   ```

3. **C# Projects**
   ```bash
   cd my_c#_project/simple_calculator
   dotnet run
   ```

4. **C++ Projects**
   ```bash
   cd my_c++_project/cheese
   g++ -o cheese *.cpp
   ./cheese
   ```
